# :page_with_curl: 생활코딩 - 리액트

## :hammer: 설치

1. **[Node.js](https://nodejs.org/en/)** 설치
2. **create-react-app 전역 설치** : `npm -g i create-react-app`
3. **리액트 프로젝트 생성** : 설치할 프로젝트로 이동 후 `create-react-app .`
